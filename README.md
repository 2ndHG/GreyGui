# GreyGui

GreyGui allows you to create graphical user interface (GUI) in MonoGame with its HTML-like syntax.

### Key Features
GreyGui aims to provide ***beautiful*** minimalism visuals with the following features.

* **Dynamic Scaling** - Elements can auto scale with its container size
* **Rounded Corner** - Create modern visuals with rounded rectangle
* **High Quality Font** - The text system uses signed distance field (SDF) technology to render sharp text at any font size
* **Explicit Control** - You, as a programer, will control and know when to Update and Draw your elements
* **Customizable** - Load your own Texture2D or just override the default drawing methods
